# 预测方法论

> 本文档为公开版本，与 [官网方法论页](https://www.jarvis-ai-club.com/#/methodology) 一致。  
> 所有概率由算法计算，AI 仅解读数据。

---

## 核心原则

1. **数学基线优先**：胜平负、比分概率均由 Poisson / ELO 引擎计算
2. **AI 只解读**：DeepSeek 研报注入结构化数据，**禁止虚构战绩或修改概率**
3. **透明可验证**：命中历史来自数据库自动结算，非手写数字

---

## 1. ELO 国家队评分

国际赛与世界杯模块使用 [eloratings.net](https://www.eloratings.net/) 的 ELO 评分。

**胜平负概率（对数 ELO 模型）：**

```
P(主胜) = 1 / (1 + 10^(-dr/400))
P(客胜) = 1 / (1 + 10^(dr/400))
P(平局) = 1 - P(主胜) - P(客胜)

dr = ELO_主 - ELO_客 + 主场加成
```

世界杯小组赛阶段，USA / MEX / CAN 主场 +55 ELO。

---

## 2. Poisson 比分模型

每场比赛双方进球数独立服从 Poisson 分布：

```
λ_team = clamp(1.30 + 0.18 × (ELO_team − ELO_opp + home_bonus) / 100, 0.15, 6.0)
```

- **BASE = 1.30**：近年世界杯场均进球基准
- **ALPHA = 0.18**：ELO 敏感度
- 若一方 ELO 高出 200 分，预期多进约 0.36 球

比分矩阵：

```
P(i-j) = Poisson(i; λ_home) × Poisson(j; λ_away)
```

胜平负概率 = 比分矩阵边际求和。

---

## 3. Poisson 深度权重（俱乐部赛事）

五大联赛使用历史特征 CSV（近 5 赛季主客场进失球均值）：

```
λ_home = f(主队主场进球, 客队客场失球, 主场加成)
λ_away = f(客队客场进球, 主队主场失球)
```

不依赖 ELO，适用于英超、西甲、意甲、德甲、法甲。

---

## 4. Monte Carlo + FIFA Bracket

2026 世界杯 104 场完整模拟：

1. 小组赛 72 场 → 按 FIFA 规则确定 32 强
2. 淘汰赛 R32 → 决赛，全程 ELO + Poisson 抽样
3. 运行 N 次（1k / 10k / 50k / 100k 可选）统计冠军概率

非穷举第三名组合，按真实 FIFA 晋级树推进。

---

## 5. DeepSeek AI 研报

Prompt 中注入：

- ELO 分差与胜平负概率
- 预期进球 λ_home / λ_away
- TOP 5 比分及概率
- 体彩 HAD 隐含概率（如有）

AI 输出自然语言分析，**不改变底层数学概率**。

---

## 6. 有效 ELO（动态修正）

基础 ELO 更新较慢（K=20）。为捕捉近 12 个月水平变化：

```
ELO_effective = ELO + clamp(0.20 × (ELO − ELO_一年前), ±150)
```
